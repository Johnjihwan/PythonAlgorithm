## Let's Learn about Python!
**<h3>how to 정렬</h3>**
> 정렬기초 (오름차순)
```
a = sorted([5,1,2,6])
print(a)

//result: 1,2,5,6
```
>변수에 미리 저장 후 정렬
```
a = [5,2,1,4,6]
a.sort()
print(a)
```
